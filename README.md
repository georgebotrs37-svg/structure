# 🚗 Car Dodging Game

A simple yet fun **Pygame-based car racing/dodging game** where you control a blue car and try to avoid incoming traffic while scoring points. The longer you survive, the higher your score!

---

## 🎮 Features
- Smooth **player controls** (arrow keys for movement).
- Randomly spawning **enemy cars** with increasing difficulty.
- **Start screen** and **Game Over screen** with score display.
- Dynamic **road animation** with moving lane markers.
- Simple but colorful **graphics** (cars, road, grass).
- Score tracking and difficulty scaling.

---

## 🖥️ Requirements
- Python 3.8+
- [Pygame](https://www.pygame.org/news)

Install dependencies:
```bash
pip install pygame
▶️ How to Play
1. 	Run the game:
python car_game.py
2. 	On the start screen, press any key to begin.
3. 	Use the arrow keys:
• 	⬅️ Left / ➡️ Right → Move horizontally
• 	⬆️ Up / ⬇️ Down → Move vertically
4. 	Avoid enemy cars and stay on the road.
5. 	When you crash, your score is displayed. Press any key to restart.
